

## ##烟祭web前端公共函数库

此项目为前端公共的函数库，一些常用的功能大家可以向这里面添加。个人的私有仓库中，通过`npm`安装使用。

## 基本使用

通过`npm`安装后， `import { getQueryString } from yjfl`,便可以使用了。
